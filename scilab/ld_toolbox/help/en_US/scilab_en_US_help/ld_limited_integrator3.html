ld_limited_integrator3

Implements a time discrete integrator (trapeziodal rule) with saturation of the output between min__ and max__

Calling Sequence

[sim, y] = ld_limited_integrator3(sim, ev, u, min__, max__, Ta)

Description

u * - input

y * - output

y(k+1) = sat( Ta/2 ( u(k)+u(k+1) ) + y(k), min__, max__ )
