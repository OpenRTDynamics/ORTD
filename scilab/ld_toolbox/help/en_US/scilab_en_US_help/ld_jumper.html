ld_jumper

jumper - block

Calling Sequence

[sim, out] = ld_jumper(sim, events, in, steps)

Description

out *+ - vector of size steps

in * - switching input

The vector out always contains one &quot;1", the rest is zero.

The &quot;1" moves to the right if in > 0. If the end is reached

it &quot;1" flips back to the left side
