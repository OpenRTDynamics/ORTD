ld_mute

Mute a signal based on cntrl-signal

Calling Sequence

[sim, y] = ld_mute( sim, ev, u, cntrl, mutewhengreaterzero )

Description

ev - event

u * - input

y * - output

mutewhengreaterzero - boolean parameter (%T, %F)

if mutewhengreaterzero == %T then

y = 0 for cntrl &gt; 0

y = u for cntrl &lt; 0

else

y = 0 for cntrl &lt; 0

y = u for cntrl &gt; 0
