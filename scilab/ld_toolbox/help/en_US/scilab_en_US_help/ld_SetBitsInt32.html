ld_SetBitsInt32

Copy a bit-pattern to the input

Description

in *(INT32) - input

BitPattern *(INT32)

out *(INT32) - output

BitNrStart - position in the input at which to start copying bits. Counting starts at zero

NUmBits - number of bits to copy
