ORTD Scilab Interface Toolbox >> nested

nested

  • ld_CaseSwitchNest — Switch mechanism for multiple nested simulations
  • ld_NoResetNest — Run a nested libdyn simulation and prevent resets of this simulation
  • ld_ReplaceableNest — A higher level for defining online replaceable schematics using callback functions
  • ld_WriteMemory2 — Write a portion to a persistent globally shared memory
  • ld_async_simulation — Run a nested libdyn simulation within a a thread
  • ld_global_memory — inittialise a persistent globally shared memory
  • ld_nested_exchffile — Online exchange of a nested simulation via loading *[ir].par files
  • ld_read_global_memory — Read a portion from a persistent globally shared memory
  • ld_simnest — create one (or multiple) nested libdyn simulation within a normal libdyn block it is possible to switch between them by an special input signal
  • ld_simnest2 — create one (or multiple) nested libdyn simulation within a normal libdyn block
  • ld_simnest2_replacement — create schematics that can be used as an online exchangeable simulation for nested simulations set-up using the ld_nested2 block
  • ld_statemachine — A statemachine
  • ld_survivereset — Keep stored data even if a simulation reset occurs (EXPERIMENTAL FOR NOW)
  • ld_write_global_memory — Write a portion to a persistent globally shared memory
